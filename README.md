# 🔴 Error Translator

Paste any error message. Get a plain English explanation + fix steps instantly.

**Live demo:** https://error-translator-xyz.vercel.app

## Tech stack
- React + Tailwind CSS (frontend)
- Node + Express (backend)
- Claude AI (Anthropic API)

## Run locally
\```bash
# Backend
cd backend && npm install && npm run dev

# Frontend (new terminal)
cd frontend && npm install && npm run dev
\```
